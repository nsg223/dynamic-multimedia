User Interface Design

For this project I have opted to use the Bootstrap responsive front end framework. It aligns with many of the Design principles taught in this course. Some of the key Design Principles I used through the project include:

  • White Space
  • Repetition
  • Alignment

Bootstrap uses an 1140px container for desktop environments to display the content in the center and have sufficient whitespace on either side of the container. This gives a nice and clean look to the page.

Images are aligned with equal widths in the gallery and album views. The design is repetitive and consistent making it easy to use and doesn't confuse users.

I also used the Bootswatch 'Yeti' from Bootswatch.com
